# Netflix Data Insights

This project is based on exploring and analyzing the Netflix dataset to understand patterns in content such as movies and TV shows.

I worked on cleaning the dataset, handling missing values, and then performing analysis to find useful insights like:
- Distribution of content over the years
- Most common genres on Netflix
- Country-wise content trends
- Movies vs TV Shows comparison

## What I used
- Python
- Pandas for data handling
- Matplotlib / Seaborn for visualization

## What I learned
While working on this project, I improved my understanding of:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualizing data in a meaningful way

## Output
The project includes graphs and observations that help in understanding how Netflix content has evolved over time.

## Note
This is a beginner-friendly data analysis project created for learning and practice purposes.
